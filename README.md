# Cybersecurity Homelab

A repository documenting my practical cybersecurity homelab, built using my own test environment. The project focuses on three areas: **SOC**, **IAM**, and **Cloud DevSecOps**.

The goal of this repository is to showcase hands-on work with security tools, environment configuration, event analysis, and documenting results in the form of technical reports.

## Project Scope

### SOC
Labs focused on monitoring, detection, and analysis of security events.

### IAM
Labs focused on identity and access management.

### Cloud DevSecOps
Labs focused on configuration security and container security.
